Dune: Fremen's Rising


Dune: Fremen's Rising is an RTT game where you can play as three different characters, each one with different and unique abilities.
In this adventure you will need to fight against the Harkonnen in three different scenarios and manage to defeat Rabban and come out victorious.

This game was made with a custom engine named Ragnar Engine which was expanded afterwards in order to fit the project needs.

My main contributions to this project are:

  • Cel Shading and Meshes Outline via post processing.
  • Real time lightning and shadow mapping.
  • C# scripting engince core bindings and components migration from C++ to C#.
  • Stencil system to see ocluded meshes using the stencil buffer.
  • Normal mapping.
  • Emissive colors.
  • Particles' batch renderer.
